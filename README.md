- 图片文字识别结果自动存档（历史记录）

# 目前调研已经实现功能

- 电子签名功能
    1. 使用opencv将图片灰度处理
    2. 使用opencv进行二值化处理
    3. 使用opencv进行通道背景透明处理

- 自动巡边裁剪功能
    1. 使用opencv进行图像多主体轮廓检测：分别得出（x,y,width,height）
    2. 默认取宽高最大的主体轮廓进行框选
    3. 裁剪框选框8点手势可进行拖拽，进行精修
    4. 用户确人后裁剪框内图片输出
